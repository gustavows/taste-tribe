# 🍽️ TasteTribe

A full-stack web app for foodies to discover restaurants, browse dishes, and share reviews, with data visualizations.

Built for **ISM6225 - App Dev for Analytics** | USF M.S. in Artificial Intelligence & Business Analytics

https://tastetribe-awcza6hug2fqcjb5.canadacentral-01.azurewebsites.net/ 

[![Live Demo](https://img.shields.io/badge/Demo-TasteTribe-green)](https://tastetribe-awcza6hug2fqcjb5.canadacentral-01.azurewebsites.net/)

---

## ✨ Features

- ➕ Create, read, update, and delete restaurant reviews
- 🔍 Find and explore restaurants by cuisine type
- 📊 Data visualizations — average ratings, cuisine popularity, and review trends
- ☁️ Deployed on Microsoft Azure

## 🛠️ Tech Stack

- **Backend:** ASP.NET Core MVC (C#)
- **Database:** Azure SQL Server + Entity Framework Core
- **Frontend:** Razor Views, Bootstrap, Chart.js
- **Deployment:** Azure App Service
